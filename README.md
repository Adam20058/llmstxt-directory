# llmstxtdirectory.org

A public, community-maintained directory tracking adoption of the `llms.txt` standard.

This project aims to:
- List websites that have published a valid `llms.txt` file
- Provide simple verification tooling
- Help businesses generate a compliant `llms.txt` file

## What is llms.txt?

`llms.txt` is a proposed standard that allows website owners to express how their content may be used by large language models.

This project is **not affiliated** with the llms.txt specification.

## Project scope

- Public directory of verified `llms.txt` files
- Simple verification logic
- Optional llms.txt generator
- No accounts
- No monetization
- No tracking beyond basic aggregate metrics
